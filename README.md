# Trabook Travel Agency Landing Page

A beautiful, premium, and fully responsive travel agency landing page built using plain HTML and CSS. This project was recreated from a meticulously designed Figma concept, focusing on modern web design principles, fluid responsiveness, and interactive hover states.

## 🚀 Features

- **Modern & Premium UI:** Incorporates a vibrant color palette, soft drop shadows, and curved elements to match modern design standards.
- **Fully Responsive:** Adapts seamlessly across desktop, tablet, and mobile devices using CSS media queries.
- **Hero Section:** Features a dynamic layout with floating decorative badges and an absolute-positioned quick search bar.
- **Interactive Cards:** Deal and Vacation Plan cards include micro-animations on hover (lifting up, color transition).
- **Testimonial Slider UI:** Replicates a stacked layout for client reviews.
- **Custom Typography:** Utilizes Google Fonts (*Volkhov* for elegant headings and *Poppins* for clean body text).

## 🛠️ Built With

- **HTML5** (Semantic structure)
- **CSS3** (CSS Variables, Flexbox, CSS Grid)
- **Google Fonts** (Volkhov, Poppins)
- **Font Awesome 6** (Icons)
- **Unsplash/Wikimedia** (High-quality placeholder images)

## 💻 Getting Started

This project requires zero configuration or build steps. It is a completely static frontend.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ritikk00/landing-page.git
   ```

2. **Open the project:**
   Navigate to the directory and simply open `index.html` in your favorite web browser. No local server is required.
   
## 🎨 Design Reference

This implementation was inspired by a professional travel agency UI concept from the Figma community. The goal was to translate that static design into functional, responsive, and maintainable vanilla HTML/CSS code without relying on heavy frameworks like Bootstrap or Tailwind.
